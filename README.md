# HTML-Weaver 🕸️

> AI 原生 HTML 编辑器
> **进可替代 PPT/Word，退可作为现代 Dreamweaver**
> 核心思想：**改参数，不改代码**

---

## 为什么 HTML 难改？

**痛点**：领导的修改通常集中在标题、数字、模块顺序等，如果这些内容混在 HTML 代码里，每次修改都像重写一遍，效率极低。

**核心问题**：我们没有预先将"哪些东西以后一定会改"显性化。

---

## 解决方案：渐进式三阶体验

### Level 1: 入门模式（新手友好）
```
输入主题 → 选择风格 → AI 生成 → 简单编辑 → 一键导出
```
- **一步生成**：输入主题，3 秒生成完整页面
- **直接编辑**：点击文字即可修改，无需代码
- **风格预设**：商务/创意/极简/醒目，一键切换
- **智能导出**：HTML、PDF、打印、分享卡片

### Level 2: 高级模式（精细控制）
- **大纲树编辑**：Tab/Enter 快速构建结构
- **配置面板**：调整布局、间距、动画、主题
- **版本管理**：保存配置版本，支持 A/B 测试
- **实时预览**：所见即所得

### Level 3: 专业模式（完全自定义）
- **代码编辑器**：直接编辑配置 JSON
- **自定义逻辑**：注入 CSS/JS
- **批量操作**：多页面管理
- **API 集成**：程序化生成

## 使用流程

### 场景 1：快速制作汇报页面（入门模式）

1. 点击 **"AI 生成"** 或按 `Cmd+N`
2. 输入："Q4 季度销售汇报"
3. 选择风格：**商务**
4. 选择用途：**汇报**
5. 点击 **生成**，3 秒后看到完整页面
6. 点击任意文字直接修改
7. 点击 **导出** → 下载 HTML

### 场景 2：精细调整布局（高级模式）

1. 切换至 **高级模式**
2. 左侧面板：大纲树编辑器
   - `Tab` = 缩进（子区块）
   - `Shift+Tab` = 取消缩进
   - `Enter` = 新建同级区块
3. 选中区块 → 右侧面板调整配置
   - 布局：全宽/分栏/网格
   - 间距：紧凑/正常/宽松
   - 动画：淡入/滑入/缩放
4. 实时预览自动更新

### 场景 3：多版本管理（专业模式）

1. 切换至 **专业模式**
2. 编辑配置后，点击 **保存版本**
3. 输入版本描述："初稿"
4. 继续修改，再次保存："领导反馈 v2"
5. 可随时回滚到任意版本

## 核心设计原则

### 1. 改参数，不改代码
```
传统：修改标题 → 打开 HTML → 找到位置 → 修改 → 保存
Weaver：点击标题 → 直接编辑 → 自动更新配置
```

### 2. 配置分离
```
index.html    # 页面壳（很少改动）
config.js     # 可编辑配置（高频修改）
data.js       # 业务数据
theme.css     # 视觉主题
```

### 3. 渐进式复杂度
- 新手：只需输入主题
- 进阶：调整配置面板
- 专业：直接编辑代码

## Block 类型

| Block | 用途 | 配置项 |
|-------|------|--------|
| **Hero** | 首屏大标题 | 布局、标题、副标题、背景 |
| **Features** | 功能展示 | 网格数、图标、描述 |
| **Stats** | 数据统计 | 数字、单位、变化趋势 |
| **Gallery** | 图片展示 | 布局（网格/轮播）、图片比例 |
| **Text** | 富文本内容 | 对齐方式、段落样式 |
| **CTA** | 行动召唤 | 按钮文字、链接、样式 |

## 技术栈

- **框架**: Next.js 15 + React 19 + TypeScript
- **样式**: Tailwind CSS 4
- **动画**: Framer Motion
- **图标**: Lucide React
- **状态管理**: React Hooks + LocalStorage
